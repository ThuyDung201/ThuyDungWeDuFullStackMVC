#Lệnh git
1. Clone project (repo) về máy
    git clone <url>
2. Upload source lên git
   1)   git add .
   2)   git commit -m "<noidungupload>"
   3)   git push
3. Pull (sync) repo
 - git pull
4. Xoá floder / file
  1) rm -r <floder/file>
  2) git rm -r --cached <floder/file>
  3) git commit -m "<noidungxoa>"
