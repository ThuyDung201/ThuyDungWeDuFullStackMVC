#Lệnh git
1. Clone project (repo) về máy
    git clone <url>
2. Upload source lên git
   -   git add .
   -   git commit -m "<noidungupload>"
   -   git push
3. Pull (sync) repo
 - git pull
4. Xoá floder / file
  - rm -r <floder/file>
  - git rm -r --cached <floder/file>
  - git commit -m "<noidungxoa>"
